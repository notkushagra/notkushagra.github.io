Projects

Project #1

SoldOut

This is an android application that helps the students on Campus to sell and auction their goods. The application is full fledged with email and Google Auth. Additional services such as in-app notifications to notify the user of purchases and increased bids on products.

Java, Android Studio, Firebase

Project #2

Shakespearean search

A search engine that inputs boolean queries and returns relevant Shakespearean plays. The search engine is built using python and required the implementation of inverted-index data structure to store data and implemented wildcard query handling and spelling correction.

Python, nltk

Project #3

iPearl Website

A website built for Intracellular Parasite Education And Research Labs (iPEARL) at BITS Pilani, Hyderabad Campus. The lab is led by the Head Of Department, Biological Sciences. The website utilises ReactJS for the frontend and Firebase for storage, authentication and hosting services.

ReactJS, Firebase

Project #4

RL for Autonomous Driving

This is a reinforcement learning agent that learns to drive a racecar around a randomly generated track. The agent uses the Proximal Policy Optimization (PPO) algorithm implementation of Stable-Baselines3. The simulation and testing was carried out using OpenAI gym.

Python, OpenAI Gym, Stable-Baselines3

Other Notable Projects

Chatbot

The project involves developing an AI-powered conversational agent that can simulate human-like interactions. The natural language processing (NLP) model is based on seq2seq architecture, which has been trained using the Cornell Movie Dialog Corpus dataset.

Python, TensorFlow

LipNet

The project involves training a machine learning model to convert lip movements from video to text. The methods described in Yannis M. Assael's "LipNet: End-to-End Sentence-level Lipreading" were modified and implemented to achieve this goal.

tensorflow, opencv-python, imageio, gdown, matplotlib

Memory Game

A memory game built using React and Firebase is a fun and interactive way to challenge your memory skills. The game is designed to display a grid of cards with different images on them. The objective of the game is to match pairs of cards with the same image.

ReactJS, Firebase, CSS3
